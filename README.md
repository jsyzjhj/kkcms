# KKCMS自动采集影视程序

# 特别说明：
  程序开发环境：
    apache2.0  php5.6.40  mysql5.5
    程序建议使用环境：
  linux系统使用nginx1.12及以上，
  Windows系统使用apache 1.8以上
  php统一使用php5.6版本
  mysql使用5.0及以上版本


# 其他说明：
  更多详细的使用方式我已封装成开发文档，直接在里面查看搜索你的问题即可，在线文档地址：https://www.showdoc.cc/kkss?page_id=435681754337319
